# Solo STT

Голосовой ввод для macOS. Зажал клавишу — сказал — отпустил — текст появился. Работает без интернета, всё распознавание на твоём Mac.

> *Speech-to-text for macOS. Hold a key, speak, release — text appears. Runs locally, no internet required.*

<img src="Resources/Ava.jpg" width="120" align="right" />

## Как это работает

Solo STT живёт в menu bar. Зажимаешь горячую клавишу (по умолчанию — правый Option), говоришь, отпускаешь. Текст вставляется туда, где стоит курсор — в любое приложение.

Под капотом — [whisper.cpp](https://github.com/ggerganov/whisper.cpp): быстрое распознавание речи прямо на устройстве. Модель скачивается один раз (~1.5 ГБ для medium) и дальше всё работает оффлайн.

Если хочется точнее или быстрее — можно подключить облачное распознавание через OpenAI или Groq. API-ключи хранятся в Keychain.

## Установка

### Вариант A: скачать готовое

Скачай `Solo_STT.dmg` из [Releases](../../releases), открой и перетащи в Applications.

Приложение не подписано сертификатом Apple Developer, поэтому macOS заблокирует первый запуск. Разблокировать можно любым из способов:

- Правой кнопкой по `Solo STT.app` → **Открыть** → подтвердить
- Или в терминале:
```bash
xattr -cr "/Applications/Solo STT.app"
```
- Или так (то же самое, другая запись):
```bash
xattr -cr /Applications/Solo\ STT.app
```

### Вариант B: собрать из исходников

Нужен macOS 15+ и Xcode Command Line Tools.

```bash
git clone https://github.com/tmasolo0/MacOS_Voice_Input.git
cd MacOS_Voice_Input
swift build -c release
bash build-app.sh
```

Приложение окажется в `/Applications/Solo STT.app`.

## Первый запуск

### Разрешение Accessibility

Solo STT вставляет текст через эмуляцию клавиатуры — для этого нужен доступ Accessibility.

**System Settings → Privacy & Security → Accessibility** → добавь Solo STT и включи галочку.

Без этого текст не будет вставляться.

### Разрешение на микрофон

При первой записи macOS спросит разрешение на микрофон — просто нажми «Разрешить».

## Горячие клавиши

По умолчанию: **правый Option** (push-to-talk).

Можно переназначить в настройках — на любую клавишу, модификатор или F-клавишу. Нажми «Записать» в настройках и нажми нужную клавишу.

## Настройки

Открываются через меню в menu bar → **Настройки** (⌘,).

**Провайдер распознавания:**
- Локальный (whisper.cpp) — без интернета, приватность
- Облачный (OpenAI / Groq) — точнее, нужен API-ключ
- Свой сервер — для тех, кто хостит whisper сам

**Модель** (для локального режима):
- Small (~460 МБ) — быстрее, менее точная
- Medium (~1.5 ГБ) — рекомендуемая, хороший баланс

**Языки:** русский, английский, немецкий, французский, испанский, украинский или авто-определение.

**Словарь:** можно выбрать пресет (IT, дизайн, бизнес) или ввести свои термины — помогает распознавать специфические слова.

**Микрофон:** системный по умолчанию или конкретное устройство. Есть авто-нормализация громкости — полезно для Bluetooth-наушников.
